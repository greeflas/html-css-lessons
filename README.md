Photoshop Usage
---------------

* `Ctrl + 1` - default layout scale

* `Ctrl + H` - show/hide grid

* `Ctrl + '+'/'-'` - change scale of layout

* `Tab` - layout preview

* Select tool 1, `Ctrl + Alt + Left mouse button` - find element in layers

* Select linear tool, hold `Shift` and pull
